
大家好：


這份資料，是我在2018年9~12月，在陽明大學教授人工智慧概論所使用的教材。
這門課是大二的選修課，修課同學都是醫學相關科系的同學。

因為這門課是屬於概論的課程，所以內容以應用和實做為主， 
理論的部份，講的比較少。因此，蠻適合非理工科系的同學，或初學者使用。

非常感謝台灣人工智慧學校的老師們和同學們，由於他們的殷勤教導和相互鼓勵， 
引導我在2018年5~8月進入AI的殿堂，也才有這份資料。

也要感謝林孟翰、洪佳聖、林岳欣、黃瑋倫擔任這門課的助教，協助準備教材，
以及回答同學們的問題。

每一件事情，都需要很多人的努力，才能完成。在感激之餘，希望這份資料，
